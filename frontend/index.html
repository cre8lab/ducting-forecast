Physics Reference

Refractivity N (ITU-R P.453)

N = 77.6·P/T + 373246·e/T²

P=hPa, T=K, e=vapor pressure (hPa)

Modified Refractivity M

M = N + (h/Rₑ)·10⁶

Ducting: dM/dh < 0 (dN/dh < -157 N/km)

Critical Frequency

f_c = c/(2h·√(2|ΔN|·10⁻⁶))

n41 (2.5 GHz) needs h > ~50m duct

Max Hop Distance

θ_c = √(2|ΔN|·10⁻⁶) rad d = 2·h / θ_c km

Ducted n41: 200–600+ km typical
